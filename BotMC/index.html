How To Install

Make Sure You Have Node And Git Installed Already!


  • 1.Open A Command Promt and locate it to folder where your bot want to installed on
  • 2.Now, Type : "git clone https://github.com/MoonLGH/NormalMineflayerBot.git" In The Command Prompt
  • 3.Now Move Command prompt to the new Directory, Then Type "npm i" in your command promt
  • 4.After npm installation done, Setting Your Bot
  • 5.After Done, Copy Json and paste it into your empty config.json
  • 6.All settings are done, now you can run the bot by typing, "node ." in the command promt
